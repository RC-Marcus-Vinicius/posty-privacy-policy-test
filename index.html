Política de Privacidade do Posty - Ambiente de Testes

Este aplicativo foi desenvolvido exclusivamente para fins de testes e desenvolvimento interno.

As informações solicitadas durante o processo de autenticação, como nome, e-mail e foto de perfil, são utilizadas apenas para validar a integração com a plataforma Meta (Facebook).

Nenhum dado pessoal é armazenado, compartilhado ou utilizado para qualquer outro fim além do funcionamento da autenticação durante o teste.

Ao utilizar este aplicativo, você concorda que seus dados serão utilizados apenas temporariamente para validar o funcionamento e serão descartados assim que o teste for finalizado.
